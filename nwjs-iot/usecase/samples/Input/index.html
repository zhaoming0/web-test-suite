This sample demonstrates picker controls for date, telephone, email and URL input types.

Type "h" in the input box to see a selection list displayed.

1. Type "a", choose the first value in the selection list, one email address begin with 'a' letter displayed in the input box

2. Then type ",a", choose the second value in the selection list, two email addresses begin with 'a' letter displayed in the input box.

The value of the input text should be "012".

The date picker can select year/month/day.

This sample is designed by HTMLInputElement type attribute.
